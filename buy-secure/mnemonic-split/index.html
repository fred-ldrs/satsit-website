2/3 Mnemonic Split

3 Backups, 3 Orte — 2 davon reichen zur Wiederherstellung. 1 verloren? Kein Problem. 1 gefunden? Kein Risiko.

Seed Security

What is a 2/3 Mnemonic Split?

Beim 2/3 Mnemonic Split werden aus deiner 24-Wörter Seed Phrase drei Backups erstellt. Jedes Backup enthält einen überlappenden Ausschnitt der Wörter — sodass je zwei Backups zusammen die vollständige Seed Phrase ergeben. Jedes Backup wird an einem anderen, sicheren Ort aufbewahrt.

Geht ein Backup verloren, kannst du mit den verbleibenden zwei deine Wallet jederzeit wiederherstellen. Findet jemand ein einzelnes Backup, hat er keine Möglichkeit, daraus die vollständige Seed Phrase zu rekonstruieren.

Ort 1

Backup A
z. B. Zuhause im Safe

Ort 2

Backup B
z. B. Bankschließfach

Ort 3

Backup C
z. B. vertraute Person

Wiederherstellung — welche Kombinationen funktionieren:

  • A + B  → vollständige Seed Phrase
  • A + C  → vollständige Seed Phrase
  • B + C  → vollständige Seed Phrase

Ein einzelnes Backup (A, B oder C allein) reicht nicht aus.

  Vorteile auf einen Blick

  • Redundanz: Ein verlorenes Backup ist kein Problem — die anderen zwei reichen zur Wiederherstellung.
  • Sicherheit: Ein einzelnes gefundenes Backup ist wertlos — kein Angreifer kann damit etwas anfangen.
  • Drei Standorte: Verteile die Backups auf drei verschiedene, sichere Orte.
  • Keine Software nötig: Stift, Papier und das Template reichen völlig aus.

Bewahre niemals zwei Backups am selben Ort auf. Die Sicherheit des Verfahrens steht und fällt mit der räumlichen Trennung der drei Backups. Teste die Wiederherstellung unbedingt, bevor du deine echte Seed Phrase auf diese Weise sicherst.
